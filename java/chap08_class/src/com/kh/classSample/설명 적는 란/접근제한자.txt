접근제한자
	클래스, 메서드, 변수 등의 멤버에 대한 접근 권한을 제어하는데 사용합니다.
	다른 클래스나 패키지에서 해당 멤버에 접근할 수 있는지 여부를 결정.
	자바에서는 네가지 주요 접근 제한자가 있음.
	
** = 이거 두개만 씀	
**	public : 어떤 클래스에서든 접근이 가능
	protected : 같은 패키지 내에서는 접근 가능하고,    // extends 를 통해서 쓸수 있음
				다른 패키지에서는 해당 클래스를 상속한 하위 클래스만 접근 가능
	default : 같은 패키지 내에서만 접근이 가능
				접근 제한자를 명시하지 않으면 default 적용!
**	private : 같은 클래스 내에서만 접근이 가능

package란 어떠한 폴더 위치를 작성하는 공간
내가 만들고싶은 파일 경로(폴더 위치)를 설정할 수 있음

자바 명명 규칙

자바의 명칙 규칙은 클래스나 패키지 변수 상수 메소드 등 
자바 프로그래밍을 할 때 이름을 따라할 규칙

카멜케이스 camelcase(낙타에 비유함)

이름에 여러 단어를 혼합해서 사용한다면 첫 번재 단어는 소문자, 두 번째 단어부터는 대문자로 시작 
ex)	helloWorld
	myName
	firstName
	
이름을 지을 때 숫자를 먼저 입력하면 안됨. 변수와같음

스네이크 케이스
모든 문자의 첫 자는 소문자로 표기하고 문자들 사이에는 _(언더바)로 연결 
ex)	name_01
	first_name
	phone_number
	
파스칼 케이스

모든 문자의 첫 자를 대문자로 표기 중간에 대문자를 중간에 넣어 표기가능.
ex)	FirstName
	PhoneNumber
	
케밥 케이스

모든 문자의 첫 자는 소문자로 표기하고 문자들 사이에는 -(하이바)로 연결
ex)
	first-number
	last-number
	phone-number
